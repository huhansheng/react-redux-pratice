 基于create-react-app小脚架搭建的react+redux框架  
----
技术栈
----  
react + react-router + redux +superagent  

项目运行
----  

	npm install  
	npm start 启动  
	npm run build 部署
	npm lint 代码格式化 


项目结构 
---- 
>api 接口路径、json-server模拟接口、、、  
common 公共模块  
component 组件  
entries 入口文件  
middleware redux中间件  
reducer reducer管理 
store state的管理  
router  路由管理  
util 公共处理   
view  界面
